# 二人捕鱼 GameLine Overview

## 1. 游戏项目定义
- 二人捕鱼是一个 GameLine（游戏项目）。
- 该游戏项目属于持续发炮型博彩玩法。
- 不以传统“胜负”作为核心描述，而以持续消耗与持续收益循环为主。

## 2. 核心玩法目标
- 玩家通过持续发炮击杀鱼，获得金币回报。
- 只要玩家仍满足发炮条件，就可以持续游玩。

## 3. 持续发炮条件
- 每一发子弹都需要消耗金币。
- 玩家身上携带的金币必须足以支付当前这一次发射动作所需金币值，才可以发炮。

## 4. 收益结算
- 简化口径下，击杀鱼后的金币回报可按：
  - `炮倍 * 鱼分倍数`
- 说明：
  - 当前仅为简化版口径
  - 后续仍有更复杂的收益公式待补充

## 5. 场次结构
- 二人捕鱼共有 7 个场次：
  - 6 个双人场
  - 1 个单人场

## 6. 场次差异维度
各场次之间的主要差异包括：
- 入场金币门槛
- 炮倍范围
- VIP 等级限制
- 等级线（Level）玩家属性限制
- 鱼池差异

## 7. 炮倍
- 炮倍范围指各场次可选的炮值区间。
- 不同场次可配置不同的炮倍范围。

## 8. 鱼池
- 各场次鱼池会不同。
- 待后续补充鱼池差异的具体体现方式。
